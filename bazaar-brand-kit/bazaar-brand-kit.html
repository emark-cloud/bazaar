The autonomous agent marketplace. A geometric identity system built on near-black and electric amber — where many independent agents converge to deal.

01Logo Concepts
Recommended
A · Convergence
Four strokes — the agents — converge on a central node: the deal, the contract, the settlement. Parent of the four persona sigils.
B · Exchange
Two offset arrows passing value in opposite directions — negotiation as give-and-take. Reads as motion.
C · Gateway "B"
A monogram B built from market-stall bars — a gate you enter. Most literal, most "marketplace."

07Usage Rules
Do
  • Amber is the only chrome accent
  • Keep clear space ≥ the central node around the mark
  • Use the mono face for all numbers, STT, hashes
  • Green/red mean value only — never decoration
Don't
  • Recolor the mark outside amber / ink / white
  • Add gradients, gloss, or drop shadows
  • Introduce a third chrome color
  • Stretch, rotate, or outline the wordmark
